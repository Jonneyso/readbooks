一、选择题

1．下列有关文学常识和名著阅读的表述，有错误的一项是（ ）
  1. A．新闻具有及时性和真实性的特点，它以消息、通讯、特写等样式，向我们提供各方面最新的资讯。报告文学强调真实，又不同新闻，作者可以对所涉及的事件和人物进行合理的艺术加工，也可以充分表达自己的思想感情。
  2. B．《大战中的插曲》是选自《聂荣臻回忆录》的一个章节，其中“大战”指历史上著名的“百团大战”，“插曲”在文中指拯救两个日本小姑娘的故事。
  3. C．《大卫·科波菲尔》是法国作家狄更斯的代表作，小说自始至终采用第一人称叙事，并采用了“纪实”的方法，小说的结局体现了狄更斯的道德信念：善恶必报。
  4. D．《复活》是托尔斯泰最后一部长篇小说，是作家一生探索和思想的总结，被誉为俄国批判现实主义发展的高峰。
【答案】
C
【解析】
C项，“法国”错。《大卫·科波菲尔》是英国小说家查尔斯·狄更斯创作的长篇小说。

一、选择题

1.《大卫·科波菲尔》的作者是（）
  1. A.狄更斯
  2. B.莎士比亚
  3. C.托尔斯泰
  4. D.雨果
【答案】
A
【解析】
《大卫·科波菲尔》是英国小说家查尔斯·狄更斯的长篇小说。莎士比亚是英国文艺复兴时期剧作家、诗人，代表作有《哈姆雷特》《罗密欧与朱丽叶》等；托尔斯泰是俄国作家，代表作有《战争与和平》《安娜·卡列尼娜》等；雨果是法国作家，代表作有《巴黎圣母院》《悲惨世界》等。
2.大卫·科波菲尔的继父是（）
  1. A.摩德斯通
  2. B.佩格提
  3. C.威克菲尔
  4. D.尤利亚·希普
【答案】
A
【解析】
摩德斯通是大卫·科波菲尔的继父，他冷酷、专横，在大卫母亲再婚后对大卫非常严厉和苛刻。佩格提是大卫家的女仆，对大卫关怀备至；威克菲尔是一位律师；尤利亚·希普是一个虚伪、阴险的人。
3.大卫在伦敦的第一份工作是（）
  1. A.在律师事务所当抄写员
  2. B.在货行做童工
  3. C.当记者
  4. D.当教师
【答案】
B
【解析】
大卫的母亲去世后，他被继父送到伦敦的货行做童工，在那里他经历了艰苦的生活。后来他才到律师事务所当抄写员，再之后成为记者，最后有过当教师的经历。
4.下列人物中，与大卫·科波菲尔感情最好的是（）
  1. A.斯提福兹
  2. B.佩格提
  3. C.米考伯先生
  4. D.阿格尼斯
【答案】
D
