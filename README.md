# 读名著 - 上海高考名著导读

## 项目介绍

本项目是一个基于HTML、CSS和JavaScript的静态网站，专门针对上海高考名著阅读要求设计，为学生提供直观、全面的名著导读服务。网站包含中国古典文学名著和外国文学名著两大类，所有名著均包含作品简介、主要人物、主题思想等内容，部分名著还包含高考知识点整理和人物故事线功能。

## 功能特点

1. **名著导读**：提供作品简介、主要人物、主题思想和结构特点等内容，支持内容筛选显示
2. **人物故事线**：部分名著支持人物分类展示和搜索
3. **人物详情页**：点击人物卡片进入详情页，查看人物的详细信息和故事时间轴（部分名著）
4. **时间线可视化**：以时间线的形式展示人物的重要事件（部分名著）
5. **高考知识点**：整理名著相关考试知识点，按照高中月考、期中考、期末考、高考分类（部分名著独有）
6. **人物关系图**：以思维导图或关系图形式展示人物关系（部分名著独有）
7. **响应式设计**：适配不同屏幕尺寸的设备
8. **完善的导航**：每个页面都有返回上一层页面的链接按钮，层级清晰

## 快速开始

1. 克隆或下载项目到本地
2. 进入项目目录
3. 启动本地服务器：
   ```bash
   python start_server.py
   ```
4. 访问 http://localhost:8091 查看项目

## 页面层级结构

项目采用两层页面结构，层级导航规则如下：

```
第一层：名著主页
    └── index.html（名著卡片）

第二层：名著内容页
    ├── 名著导读
    └── 高考知识点（部分名著独有）

层级规则：
- 所有第二层页面都从主页名著卡片链接进入
- 返回按钮统一返回主页
```

## 设计规范

### 颜色系统

| 用途 | 颜色值 |
|------|--------|
| 主色调 | #8B4513 |
| 深色调 | #654321 |
| 背景色 | #f5f5dc |
| 卡片背景 | rgba(255,255,255,0.9) |

### 按钮样式

导航按钮采用统一的卡片式设计：
- 默认状态：棕色背景 #8B4513
- 悬停状态：深棕色背景 #654321
- 活动状态：深棕色背景 #654321

## 技术实现

1. **HTML结构**：使用语义化标签构建页面结构
2. **CSS样式**：使用Flexbox和Grid布局实现响应式设计
3. **JavaScript**：使用Fetch API加载JSON数据，动态渲染页面内容
4. **关系图**：使用SVG和D3.js实现人物关系可视化（部分名著）
5. **内容筛选**：通过CSS类切换实现内容显示/隐藏
