# 读名著 - 中国古典文学

## 项目介绍

本项目是一个基于HTML、CSS和JavaScript的静态网站，用于展示中国古典文学名著，目前包含《水浒传》、《红楼梦》、《西游记》和《三国演义》四部名著。项目使用本地JSON文件存储人物数据，通过前端代码动态渲染人物信息和时间线。

## 功能特点

1. **名著导读**：提供作品简介、主要人物、主题思想和结构特点等内容，支持内容筛选显示
2. **人物分类展示**：将人物按天罡星/地煞星（或类别）分类展示
3. **人物搜索**：支持关键词搜索人物
4. **人物详情页**：点击人物卡片进入详情页，查看人物的详细信息和故事时间轴
5. **时间线可视化**：以时间线的形式展示人物的重要事件
6. **人物关系图**：以思维导图或关系图形式展示人物关系
7. **高考知识点**：整理红楼梦相关考试知识点（红楼梦独有）
8. **响应式设计**：适配不同屏幕尺寸的设备
9. **图片加载优化**：优先从本地加载人物肖像图片，失败后尝试网络加载
10. **完善的导航**：每个页面都有返回上一层页面的链接按钮，层级清晰

## 快速开始

1. 克隆或下载项目到本地
2. 进入项目目录
3. 启动本地服务器：
   ```bash
   python start_server.py
   ```
4. 访问 http://localhost:8090 查看项目

## 页面层级结构

项目采用三层页面结构，层级导航规则如下：

```
第一层：名著主页
    └── index.html（名著卡片）

第二层：名著内容页
    ├── 名著导读
    ├── 人物故事线
    └── 独有页面
        ├── 水浒传：人物聚散全景
        ├── 红楼梦：人物关系图
        └── 红楼梦：高考知识点

第三层：人物详情页
    └── 从人物故事线进入

层级规则：
- 所有第二层页面都从主页名著卡片链接进入，返回按钮统一返回主页
- 第三层页面从人物故事线的人物卡片链接进入，返回按钮返回人物列表
- 同级页面之间不设置链接跳转
```

## 人物详情页布局

人物详情页采用左右排版布局（3.5:6.5比例）：

- **左侧（35%）**：人物肖像图片 + 图片说明文字
- **右侧（65%）**：信息卡片网格
  - 第一行：绰号卡片 + 星位卡片（各占50%）
  - 第二行：别名卡片（占100%，全宽）
  - 第三行：人物简介卡片（占100%，全宽）
- **下方**：故事时间轴

## 技术实现

1. **HTML结构**：使用语义化标签构建页面结构
2. **CSS样式**：使用Flexbox和Grid布局实现响应式设计
3. **JavaScript**：使用Fetch API加载JSON数据，动态渲染页面内容
4. **图片加载**：实现本地图片优先加载，失败后尝试网络加载，最后使用默认图片作为回退
5. **关系图**：使用SVG和D3.js实现人物关系可视化
6. **内容筛选**：通过CSS类切换实现内容显示/隐藏
