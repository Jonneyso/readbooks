一、选择题

1.《呐喊》收录了鲁迅从1918年至1922年所作的14部短篇小说，以下哪一部不属于《呐喊》集？（）
  1. A.《狂人日记》
  2. B.《孔乙己》
  3. C.《祝福》
  4. D.《药》
【答案】
C
【解析】
《祝福》出自鲁迅的小说集《彷徨》，而《狂人日记》《孔乙己》《药》都收录在《呐喊》中。《狂人日记》是中国第一部现代白话小说，以独特的视角揭示了封建礼教"吃人"的本质；《孔乙己》塑造了一个深受封建科举制度毒害的知识分子形象；《药》通过描写茶馆主人华老栓夫妇为儿子小栓买人血馒头治病的故事，反映了辛亥革命前后农村的社会现实。
2.在《阿Q正传》中，阿Q所居住的村庄叫什么？（）
  1. A.未庄
  2. B.鲁镇
  3. C.咸亨酒店所在的镇
  4. D.乌镇
【答案】
A
【解析】
阿Q居住在未庄。鲁镇是《祝福》《孔乙己》等故事发生的地方；咸亨酒店是《孔乙己》中孔乙己常去喝酒的地方；乌镇与《呐喊》中的故事并无关联。未庄是一个封闭、落后、愚昧的小村庄，阿Q的悲剧命运就发生在这个特定的环境中。
3.《故乡》中，"我"回故乡的目的是（）
  1. A.为了看望久别的亲人
  2. B.为了卖老屋，接母亲到城里去
  3. C.为了寻找童年的伙伴
  4. D.为了参加亲戚的婚礼
【答案】
B
【解析】
在《故乡》中，"我"回到故乡是因为老屋已经卖给了别姓，要搬家，所以是为了卖老屋，接母亲到城里去。
4.《药》中，华老栓为儿子买的"药"是（）
  1. A.人血馒头
  2. B.中药
  3. C.西药
  4. D.仙丹
【答案】
A
【解析】
《药》中，华老栓为儿子买的"药"是人血馒头。他听信迷信，认为人血馒头可以治疗儿子的痨病，于是从刽子手康大叔那里买了蘸着革命者夏瑜鲜血的馒头给儿子小栓吃。
5.《狂人日记》中，狂人认为赵家的狗看他两眼的意思是（）
  1. A.狗也知道他是狂人
  2. B.狗是赵家的帮凶，想害他
  3. C.狗看他不顺眼
  4. D.狗只是随便看看
【答案】
B
【解析】
在《狂人日记》里，狂人处于一种极度的被害妄想状态，他认为整个世界都在吃人，赵家的狗看他两眼也是在帮赵家监视他，想害他。这反映了狂人对封建礼教的恐惧和反抗。

二、简答题

6.请简要概括《呐喊》的主题思想。
【答案】
《呐喊》真实地描绘了从辛亥革命到五四运动时期的社会生活，揭示了种种深层次的社会矛盾，对封建制度及陈腐的传统观念进行了深刻剖析和彻底否定，表现出对民族生存浓重的忧患意识和对社会变革的强烈渴望。
【解析】
《呐喊》是鲁迅的第一部小说集，收录了他从1918年至1922年创作的14篇短篇小说。这些作品通过塑造一系列生动的人物形象，如狂人、孔乙己、阿Q等，揭示了当时社会的种种问题，如封建礼教的压迫、民众的愚昧麻木、辛亥革命的不彻底性等，表达了鲁迅对社会现实的批判和对民族命运的关注。
7.《阿Q正传》中的"精神胜利法"具体表现在哪些方面？
【答案】
阿Q的"精神胜利法"主要表现在以下几个方面： （1）自欺欺人：明明被打，却说是"儿子打老子"； （2）自我安慰：被人抢了钱，却说是"算是被儿子拿去了"； （3）盲目自大：认为自己比别人强，看不起别人； （4）健忘：很快就忘记痛苦和屈辱。
【解析】
"精神胜利法"是阿Q性格的核心特征，是他在被压迫、被剥削的社会环境中形成的一种自我保护机制。这种心理特征反映了当时中国社会底层民众的精神状态，也是鲁迅对国民性弱点的深刻批判。通过阿Q的"精神胜利法"，鲁迅揭示了旧中国民众的愚昧、麻木和自我安慰的心理，以及这种心理对社会进步的阻碍。

一、单选题

1.鲁迅的《呐喊》首次出版于哪一年？（）
  1. A.1918年
  2. B.1923年
  3. C.1921年
  4. D.1925年
【答案】
B
【解析】
《呐喊》是鲁迅的第一部小说集，收录了他从1918年至1922年创作的14篇短篇小说，首次出版于1923年。1918年是鲁迅发表《狂人日记》的年份，1921年是他发表《阿Q正传》的年份，1925年是他发表《彷徨》的年份。
2.《阿Q正传》中，阿Q的"精神胜利法"主要体现了他怎样的心理特征？（）
  1. A.勤劳朴实
  2. B.自欺欺人
  3. C.勇敢无畏
  4. D.智慧机敏
【答案】
B
【解析】
"精神胜利法"是阿Q逃避现实的典型心理机制，体现了他自欺欺人的性格特征。
3.鲁迅的哪部作品被胡适誉为"中国现代文学史上第一篇白话小说"？（）
  1. A.《狂人日记》
  2. B.《孔乙己》
  3. C.《药》
  4. D.《故乡》
【答案】
A
【解析】
《狂人日记》是鲁迅于1918年发表在《新青年》杂志上的第一篇白话小说，它以独特的视角和形式，揭示了封建礼教"吃人"的本质，被胡适誉为"中国现代文学史上第一篇白话小说"。这部作品标志着中国现代白话小说的开端，具有重要的历史意义。
